# C1: 忠实的知识注入（核心发现，v3新增）

**Claim**: Cross-attention注入的Knowledge Compliance显著高于RAG的in-context注入。在反事实知识条件下，TokenMem忠实遵从注入的知识（高compliance、低conflict），而RAG因知识冲突导致输出不可靠（低compliance、高conflict）。

**前提条件**: C4（知识敏感性）必须先成立——排除"adapter太弱，什么都忽略"的零假设。

**Minimum convincing evidence**:
- TokenMem counterfactual KC ≥ 60%（模型确实跟随了反事实知识）
- RAG counterfactual KC < TokenMem counterfactual KC by ≥ 15pp
- TokenMem conflict rate < RAG conflict rate
- 效果在strong-prompt RAG基线下仍成立
- 至少在2个模型上复制

**评价指标**:
- Knowledge Compliance (KC): 模型回答与注入知识支持的答案一致的比例
- Conflict Rate: 模型回答既非ground truth也非知识支持答案的比例

**机制假说**: Cross-attention是独立于self-attention的知识通道，不与参数记忆在residual stream中竞争，因此reduced competition → higher compliance。

**Status**: ❌ Pending — 需要counterfactual compliance实验 + strong-prompt baseline

**⚠️ 评测方法更新 (2026-04-29, exp:E2_pilot_eval_method)**:

MCQ logprob评测存在严重天花板效应：vanilla_rag在cf_medqa上达94%遵从率（MCQ直答），使≥15pp差距不可能达到。但CoT评测将遵从率降至28-36%，释放出充分差异化空间。

**推荐评测协议**: MCQ + CoT（主要指标） + MCQ logprob（辅助对照）。三分类：遵从反事实 / 坚持参数化知识 / 其他。

**阈值说明**: ≥15pp 标准应基于 CoT-based KC 而非 MCQ logprob KC。MCQ logprob作为辅助参考。
