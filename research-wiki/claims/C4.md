# C4: 知识敏感性（Knowledge Sensitivity）

**Claim**: TokenMem adapter确实使用注入的知识内容（Oracle >> Shuffled ≈ Empty），排除"cross-attention仅作为正则化"或"adapter太弱忽略一切"的零假设。

**角色**: C1（忠实注入）的**必要前提条件**。没有C4，C1的高compliance可能被解释为adapter根本没在用知识。

**Minimum convincing evidence**:
- Oracle accuracy > Empty accuracy by ≥ 10pp
- Shuffled accuracy ≈ Empty accuracy (差距 < 3pp)
- 在至少2个数据集上成立

**Status**: ❌ Pending — 需要Oracle/Shuffled/Empty实验

**旧系统参考**: Memory-LoRA (0.6B) 曾得到 KS=+56.40%（Oracle远超Shuffled≈Empty），预期新系统也能复现方向。
